# @portfolio/shared-ui

Shared React components.

## Usage

```tsx
import { Ui } from '@portfolio/shared-ui';
```

## Commands

```bash
pnpm nx test ui
pnpm nx build ui
pnpm nx lint ui
```

## Add Component

```bash
pnpm nx g @nx/react:component button --project=ui
```

Then export from `src/index.ts`.
